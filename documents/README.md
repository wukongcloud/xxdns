# XXDNS 文档

## 产品简介

XXDNS 是基于 bind9 打造的一款全功能的界面化的 DNS 管理平台，支持通过 restfulapi 对提供的所有功能进行操作，简化了bind9的 配置，并提供解析日志分析，性能监控，数据备份，操作审计等功能。
